**Interactive Tableau dashboard analyzing California Healthcare-Associated Infections (HAI) by facility type, focusing on Standardized Infection Ratio (SIR) and infection rates.**  

---

## 🔍 Overview
This dashboard provides a clear view of hospital infection performance across California. It helps users:

- Compare **Standardized Infection Ratio (SIR)** by facility type  
- Explore **infection rates** for different hospitals  
- Identify facilities performing **better or worse than expected**  
- Make data-driven insights for healthcare quality improvement  

The visuals include bar charts, scatter plots, and color-coded indicators to make interpretation intuitive.

---

## 📈 Key Metrics
| Metric | Description |
|--------|-------------|
| **SIR (Standardized Infection Ratio)** | Risk-adjusted ratio of observed vs. predicted infections |
| **Infection Rate** | Raw infection count per 1,000 central line days |
| **Facility Type** | Hospital category for comparative analysis |

## Data Preparation:
- Filtered for hospitals with complete SIR and infection rate data
- Calculated average SIR by facility type
- Added color-coded performance metrics for visualization
